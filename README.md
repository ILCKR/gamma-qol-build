# Quality of Life G.A.M.M.A. 0.9.4 Modpack
Графический и интерфейсный аддон-пак для G.A.M.M.A. 0.9.4, переносящий визуальную эстетику S.T.A.L.K.E.R. 2 и современные тактические механики, максимально избегая изменений оригинального баланса. Сборка включает исправления движка, переработанную систему лечения и функции автоматизации интерфейса для повышения общего комфорта игры.

# AOEngine 0.5.2

1. [Скачать версию БЕЗ профайлера](https://discord.com/channels/1387363886654689320/1411320046038421668/1411320051570835476)
2. Переименовать "..\Anomaly\bin\ **AnomalyDX11.exe**" в "**AnomalyDX11**" (без расширения .exe)
3. Распаковать "**AnomalyDX11.exe**" из архива AOEngine по пути "..\Anomaly\bin"
4. В MO2 запускать игру через "**Anomaly (DX11)**"
